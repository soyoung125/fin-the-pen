## ⚙️ Run Locally

1. Node.js 설치를 확인해주세요.
2. yarn 설치를 확인해주세요.

```bash
npm i -g yarn
```

3. 프로젝트를 클론 합니다.

```bash
  git clone https://github.com/soyoung125/fin-the-pen-web.git
```

4. 프로젝트에 진입합니다.

```bash
  cd fin-the-pen-web
```

5. 패키지를 설치합니다.

```bash
  yarn
```

6. 서버를 실행합니다.

```bash
  yarn dev
```
